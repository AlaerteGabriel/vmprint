vmprint is a small program that helps you create high-quality PDF files. It works on many systems, including browsers and Cloudflare Workers. - Supports running in browsers or server environments.

This tool helps people who want quick typesetting without large software. You just provide content, and vmprint creates PDF files you can open anywhere.

## ▶️ Running vmprint

vmprint is not a regular program with a graphical interface. It works by running scripts that create PDF files. Here are two simple ways to use vmprint.

### Option 1: Use vmprint in the browser

1. Open the `index.html` file from the extracted folder (if it exists) by double-clicking it.
2. This opens a basic demo page in your browser.
3. The demo runs vmprint and shows PDF output you can try.

### Option 2: Use vmprint with Node.js

Use this if you want to create PDFs from the command line or a script.

1. Download and install [Node.js](https://nodejs.org/) if you do not have it.
2. Open Command Prompt:
   - Press `Windows + R` keys.
   - Type `cmd` and hit Enter.
3. Navigate to the folder where vmprint files are located by using the `cd` command. Example:
   ```
   cd C:\Users\YourName\Downloads\vmprint-x.x.x
   ```
4. Run vmprint scripts using `node`. For example:
   ```
   node vmprintDemo.js
   ```
5. This should produce a PDF file in the folder, showing vmprint's output.

## 📚 Using vmprint

Since vmprint is a typesetting engine, it works by processing commands or scripts written in JavaScript. Here is a simple overview.

- You write a small JavaScript script that tells vmprint what text to print, fonts to use, and page layout.
- vmprint reads this script and creates a PDF file based on those instructions.
- The PDF file can be opened in any PDF viewer like Adobe Reader.

You do not need to know advanced programming to use the demos. You can change some text inside demo files and run them again.
